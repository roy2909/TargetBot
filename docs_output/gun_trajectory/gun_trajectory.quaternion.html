gun_trajectory.quaternion module

gun_trajectory.quaternion.euler_to_quaternion(roll, pitch, yaw)

Convert Euler angles to a quaternion.

roll (float): Rotation around the x-axis (roll) in radians. pitch (float): Rotation around the y-axis (pitch) in radians. yaw (float): Rotation around the z-axis (yaw) in radians.

Quaternion: The quaternion representing the input Euler angles.

gun_trajectory.quaternion.quaternion_multiply(q0, q1)

Multiplies two quaternions.

Parameters

  • q0 – A 4 element array containing the first quaternion (q01, q11, q21, q31)

  • q1 – A 4 element array containing the second quaternion (q02, q12, q22, q32)

Returns

A 4 element array containing the final quaternion (q03,q13,q23,q33)
